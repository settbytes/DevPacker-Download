# DevPacker - Teoria e Foco ®

O **DevPacker** é uma ferramenta desktop de alta performance desenvolvida para facilitar a vida de desenvolvedores que precisam organizar, fazer backup ou compartilhar o contexto completo de seus projetos de código-fonte em um único local. 

Com uma interface intuitiva e moderna, o software transforma estruturas complexas de pastas em arquivos consolidados, garantindo que nenhum detalhe da arquitetura original seja perdido.

---

## ✨ Principais Funcionalidades

* **Empacotamento Inteligente:** Varre diretórios inteiros e gera um mapa visual da árvore de pastas para facilitar a compreensão do projeto.
* **Formatos Flexíveis:** Exporte seu código em `.txt` (ideal para leitura humana) ou `.json` (focado em precisão para leitura por máquinas e IAs).
* **Filtros de Segurança:** O programa ignora automaticamente pastas pesadas ou desnecessárias (como `node_modules` ou `.git`), focando apenas no que importa: o seu código.
* **Reconstrução Cirúrgica:** A aba de desempacotamento lê os arquivos gerados e recria toda a estrutura física de diretórios no seu computador.
* **Preservação de Mídias:** Para arquivos que não são texto (como imagens e vídeos), o software cria lembretes em `.txt` para que você saiba exatamente onde repor os arquivos originais após a reconstrução.

---

## 📥 Download e Instalação

O DevPacker é um software **portátil** para Windows (10 e 11). Isso significa que ele não precisa de instalação — basta baixar e rodar.

1.  Acesse a aba **Releases** deste repositório.
2.  Baixe o arquivo `DevPacker.exe` (ou a versão `.zip` caso seu navegador bloqueie o executável).
3.  Execute o arquivo para abrir a ferramenta.

> **⚠️ Aviso de Segurança:** Como o DevPacker é um software independente, o Windows SmartScreen ou o navegador Edge podem exibir um alerta de "Fornecedor Desconhecido". 
> 
> **Para prosseguir:** No navegador, clique em "Manter" (ou "Keep"). No Windows, clique em "Mais Informações" e depois em "Executar assim mesmo".

---

## 📖 Como usar a ferramenta

### 1. Para Empacotar (Criar Backup)
* Selecione a pasta do seu projeto na origem.
* Escolha onde deseja salvar o arquivo final.
* Clique em **"Iniciar Procedimento"**.

### 2. Para Desempacotar (Reconstruir Projeto)
* Selecione o arquivo `.txt` ou `.json` que você gerou anteriormente.
* Selecione a pasta de destino.
* O programa criará uma nova pasta com o sufixo `_devpacker` para garantir que seus arquivos originais nunca sejam sobrescritos ou misturados por acidente.

---

## 🔒 Segurança e Privacidade

* **Processamento Local:** Seus dados e códigos nunca saem do seu computador. O DevPacker não faz upload de nenhum arquivo do seu projeto.
* **Gestão de Foco:** Ao clicar em "Acessar" na tela de boas-vindas, o site oficial do **Teoria e Foco** será aberto. O programa aguardará 3 segundos em background antes de aparecer, garantindo que sua navegação no site não seja interrompida.
